# Privacy Boundary

This repository is synthetic by design.

Allowed in this repository:

- Source code.
- Tests.
- Documentation.
- Synthetic internal operations notes.
- Synthetic outputs generated by local demo runs, if intentionally shared.

Not allowed in this repository:

- Real operational messages.
- Personal data.
- Live operational exports.
- Local databases.
- Environment files.
- Credentials or deployment configuration.

Real deployments should keep operational data in gitignored or encrypted storage. Credentials should be provided only through the runtime environment. CI should run on synthetic fixtures and should not require network model calls.

The demo includes a synthetic-data guard test and gitleaks configuration to reduce accidental leakage risk. Those checks are supporting controls, not a substitute for human review before publishing.
